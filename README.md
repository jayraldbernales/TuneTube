## Getting Started

1. **Set Up Your Database**:

   - Open **XAMPP** and start Apache and MySQL.
   - Create a new database titled **`tunetube`** in **phpMyAdmin**.
   - Import the `tunetube.sql` file from the `db/` folder.

2. **Accounts to Use**:
